# FlowerShop - E-Commerce Web Application

# Description
This is a responsive web application developed using ASP.NET MVC and **MSSQL. The application dynamically fetches product data (flowers) from the database and lists them using a modern grid layout. Users can click on any product to view its dedicated details page

#Features
* **Dynamic Product Listing:** Displays products dynamically from the SQL database using Grid layout.
* **Detail Page Routing:** Each item has a dedicated details page fetched by its ID.
* **Modern UI:** Clean and user-friendly interface.
* **Admin Panel (Upcoming):** CRUD operations for product management will be added soon.

#Tech Stack
* **Backend:** C# / ASP.NET MVC
* **Database:** MSSQL (Structured with a `.sql` script file included in the Database folder)
* **Frontend:** HTML5, CSS3, Bootstrap

# Türkçe Açıklama
Bu proje, **ASP.NET MVC** ve **MSSQL** kullanılarak geliştirilmiş bir e-ticaret/ürün listeleme web uygulamasıdır. Çiçek verileri veri tabanından dinamik olarak çekilir ve modern bir grid yapısıyla listelenir. Kullanıcılar ürünlerin detayına tıklayarak o ürüne ait özel detay sayfasına yönlendirilir.

# Özellikler
* **Dinamik Ürün Listeleme:** SQL veri tabanındaki ürünleri Grid yapısı ile dinamik olarak sergiler.
* **Detay Sayfası Yönlendirmesi:** Her ürünün ID tabanlı çalışan kendine ait bir detay sayfası bulunur.
* **Modern Arayüz:** Temiz ve kullanıcı dostu tasarım.
* **Admin Panel (Yakında):** Ürün yönetimi için gerekli CRUD (Ekle, Sil, Güncelle) fonksiyonları yakında eklenecektir.
